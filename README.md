#### [Portfolio Link](https://devinleejones.github.io/portfolio)

# Technologies

- JavaScript
- React.js
- HTML5
- CSS3
- BootStrap
- Material UI
- WOW.js
- Animate.css

# Usage

Clone and install dependencies.

```
git clone https://github.com/devinleejones/portfolio.git
cd portfolio
npm install
```

Create an '.env' file in the project root.

```
PORT=3000
```

Run the application

```
npm run start
```
